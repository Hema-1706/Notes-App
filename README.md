# Notes App

A responsive Notes Application built using HTML, CSS, and JavaScript.

This project focuses on advanced DOM manipulation, CRUD functionality, dynamic rendering, and browser storage handling.

---

## Features

- Add notes
- Edit notes
- Delete notes
- Save notes permanently using localStorage
- Responsive card-based UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## JavaScript Concepts Learned

- Arrays & Objects
- CRUD Operations
- Dynamic DOM Rendering
- Event Handling
- localStorage
- JSON Methods
- prompt()
- find()
- Date & Time Handling
- State Management Basics
